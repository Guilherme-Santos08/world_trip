## 🛠️ Installation

1. Clone the repository

```bash
git clone https://github.com/Guilherme-Santos08/world_trip.git
```

2. Change the working directory

```bash
cd github-profile-readme-generator
```

3. Install dependencies

```bash
yarn

or

npm install
```

4. Run the app

```bash
yarn dev

or

npm run dev
```

🌟 You are all set!
